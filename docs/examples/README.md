# examples

This folder contains example splits you can copy.

Right now it includes:

- a data-plus-frontend split
- a three-agent product split
- a reusable master prompt

Use these if you want to move quickly instead of starting from scratch.

Suggested order:

1. start with the two-agent example if you only want to understand the pattern
2. move to the three-agent example when the repo has clear product, frontend, and quality lanes
3. adapt the role boundaries to your own project instead of copying them blindly
